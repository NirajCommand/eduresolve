# Title
EduResolve. Smart Academic Interaction and Resolution System

## Problem statement
Students face unresolved academic doubts. Teachers forget context. Authorities lack visibility.

## Objective
Provide a centralized system for academic issue tracking, quizzes, assignments, and reporting.

**Users**
• Student
• Teacher
• HOD or Principal
